Presentacion de los requisitos

Se presentan los campos necesarios para hacer las instalacion de macOS en los diferentes Sistemas Operativos desde la creacion de usb y los diferentes programas necesarios

Creacion USB de boot.

exite dos formas de crear USB de boot para arranque. una es usar la imagende del sistema operativa completa para equipos que no tiene conexion a internet. si el equipo cuente con la tarjeta compatible con el sistema operativa se puede usar una imag de 500 MB para descargar.

Sin conexión

Solo se puede hacer en macOS. Windows/Linux no tiene los controladores APFS/HFS necesarios para montar un instalador completo

conexión

Se puede hacer en macOS/Linux/Windows Requiere una conexión a Internet que funcione a través de un adaptador de red compatible con macOS en la máquina de destino

Procedimientos de la instalacion

Compatible con OS X 10.4 a la actual y Soporta instalaciones heredadas y UEFI

Compatible con OS X 10.7 a la corriente, Solo instalador en línea y Soporta instalaciones heredadas y UEFI
